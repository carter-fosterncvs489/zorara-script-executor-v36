## Overview

Zorara is a Windows-oriented Roblox script executor for users who want a focused desktop environment for Lua-based scripting. It brings together script execution, a built-in hub, and an interface that keeps the main workflow straightforward and easy to navigate.

The app is centered around practical scripting tasks like one-click Lua injection, batch execution, and a persistent session queue. With auto-update support and a lightweight UI, Zorara is designed to stay lean while still giving quick access to a broad script hub with many popular Roblox script categories.

## FAQ

**Is Zorara limited to Windows?**  
Yes. The documented target platform for this build is Windows 10/11.

**Does it remain compatible after Roblox updates?**  
An auto-update engine is included to help the build stay aligned with changes, though compatibility can still shift over time.

**What sets Zorara apart from a simple script loader?**  
It includes a script hub, batch execution, session queue management, and debugger support instead of only a single load action.

**Can using a script executor create account risk?**  
Any third-party scripting tool can involve platform or account-level risk. Use it in line with Roblox terms and your own judgment.

**Where are scripts kept?**  
The product profile points to persistent session handling and a local hub structure, so script-related data is managed on the device.
